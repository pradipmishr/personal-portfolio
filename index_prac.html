FullStackNoteApp

This is a full-stack note-taking application built using Django (DRF) for the backend and ReactJS for the frontend. It features secure user authentication with JWT, allowing users to register, login, and logout. The application enables seamless CRUD operations on notes, including creating, reading, updating, deleting, pinning, and unpinning. Additionally, it supports token management, automatic token refresh, and blacklist handling for enhanced security. The backend is deployed on AWS EC2 using Gunicorn and Nginx with DuckDNS for domain management and Let's Encrypt SSL for secure HTTPS access. The frontend is hosted on Netlify for fast and scalable deployment, while the database is managed on NeonDB for high availability and performance.
